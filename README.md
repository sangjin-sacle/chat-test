## 📌 주요 기능
- **Hello World 페이지 → 챗봇 테스트 Playground로 개조**
- ChannelIO 기본 위젯 자동 부팅 (`pluginKey` 사용)
- Playground Controls:
  - 위젯 열기/닫기/토글
  - 이벤트 트래킹 (track)
  - 사용자 프로필 업데이트 (updateUser)
  - 가짜 memberId 지정
  - 세션 종료 (shutdown) & 재부팅 (boot)
- Log Console 제공 → 버튼 동작 결과 실시간 확인 가능
